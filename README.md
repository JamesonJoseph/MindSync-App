# 🧠 MindSync – AI Personalized Life Management App

MindSync is an **AI-powered personalized life management mobile application** designed to help users manage their daily life while also monitoring their mental well-being.

The application integrates multiple everyday tools such as **journaling, task management, budget tracking, reminders, and calendar features** into a single platform. Using **AI-based sentiment analysis, emotion detection, and conversational AI**, the system provides an **early warning system for mental health awareness**.

---

## 🚀 Features

### 🤖 AI Avatar Assistant (NEW)
- Interactive **AI Avatar** users can talk with
- Supports **Speech-to-Text (STT)** for voice input
- Uses **Text-to-Speech (TTS)** for voice responses
- Provides emotional support and conversational interaction
- Allows users to communicate naturally instead of typing

---

### 🧠 AI Mental Health Early Warning System
- Daily journaling feature
- AI analyzes journal entries using **LLM-based sentiment analysis**
- Detects emotional patterns over time
- Provides supportive suggestions and insights

---

### 📸 Facial Emotion Detection
- Users can take a **selfie check-in**
- AI detects facial emotions (happy, sad, neutral, etc.)
- Provides mental wellness feedback.

---

### 📝 Daily Journal
- Write daily thoughts and activities
- Track emotional trends
- Visual indication for missed journal days

---

### ✅ Task Management
- Create, edit, and delete tasks
- Set reminders for important activities
- Receive notifications before deadlines

---

### 💰 Budget Management
- Reads **bank transaction SMS messages**
- Detects credit and debit transactions
- Automatically updates balance
- Alerts when the balance becomes lower than the average

---

### 📅 Smart Calendar
- Manage events and schedules
- Add important dates and birthdays
- Receive event reminders

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/MindSync-App.git
```

Navigate into the project folder:

```bash
cd MindSync-App
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

The goal of MindSync is to create a **single intelligent system that integrates productivity tools with AI-powered mental wellness monitoring**, helping users maintain both **personal productivity and emotional well-being**.

The addition of an **AI Avatar assistant** allows users to communicate naturally with the system using voice interaction.
